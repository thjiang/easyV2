## install

```bash
npm install easyv2 -g
```

## usage

```
Usage: v2 [options] [command]

Options:
  -v, --version  output the version number
  -h, --help     output usage information

Commands:
  status|s       show v2ex status
  hot|h          show top 10 topics
  latest|l       show latest topics
  view|v         view a post
```
